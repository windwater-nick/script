# Docker 相關腳本
1. docker_check.sh
> 功能：**Docker 孤兒目錄掃描工具** 用於檢查 Docker 資料目錄是否仍被目前執行中的容器使用。自動比對容器名稱與目錄名稱，快速找出未掛載的孤兒目錄。
> 
> 適用範圍：
> * 容器名稱與資料夾名稱相符。 (假設容器名稱:abc，而資料夾名稱:edf。則會顯示 edf 未掛載)
> * 只配對當前正在運行的容器。
2. docker_ps.sh
> 功能：**Docker 容器狀態檢視腳本** 簡化 **docker ps** 輸出，提升終端機可讀性，查看 **Docker** 容器資訊，依名稱排序並**顯示容器 ID**、**名稱**、**運行狀態**與**Port**。以表格對齊及標頭輸出。
